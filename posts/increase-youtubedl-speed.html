Increase Youtubedl Speed

28-12-2021

From last month or slow, for some reason, the speed of downloading youtube videos via youtube-dl has reduced substantially. Iâ€™m receiving no more than 50 Kb/s. Many have shared the same problem on different forums. Till it is fixed by youtube-dl team, We can try a small fix. The solution here is to use Aria2c. Now what is Aria2c? It is a small command line program which increases your download speed. I may be wrong but as far as i understand it makes multiple connections to a particular server to download a file, in this way the speed can be increased by more than 4 times. (Maximum speed depends on your internet connection.)

Now the way you usually download a video using youtube-dl is this:

youtube-dl -f best https://www.youtube.com/watch?v=dQw4w9WgXcQ

But this way you are probably receiving less speed. So try this:

Note: Download Aria2 before using this. (In debian repos the program is available by the name of Aria2 not Aria2c)

youtube-dl --external-downloader aria2c --external-downloader-args '-c -j 3 -x 3 -s 3 -k 1M' -f best https://www.youtube.com/watch?v=dQw4w9WgXcQ

Using the above way you will receive the increased speed. The way we achieved it was simply using Aria2c as external downloader for youtube-dl.
